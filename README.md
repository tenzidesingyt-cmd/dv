# EnglishDV Telegram Bot

Телеграм-бот для изучения английского языка с AI-проверкой домашних заданий и аудио-уроками.

## Структура проекта

```
тг бот/
├── main.py              # Главный файл бота (точка входа)
├── requirements.txt     # Зависимости Python
├── .env.example         # Пример переменных окружения
├── Procfile             # Конфигурация для Railway/Heroku
├── railway.json         # Конфигурация для Railway
├── runtime.txt          # Версия Python
└── src/
    ├── __init__.py      # Пакет бота
    ├── config.py        # Конфигурация (токены, API ключи)
    ├── database.py      # Работа с SQLite БД
    ├── lessons.py       # Данные 25 уроков
    ├── keyboards.py     # Клавиатуры бота
    ├── ai.py            # AI промпты и функции
    └── reading_texts.py # Тексты для аудио-уроков
```

## Установка (локальная разработка)

### 1. Клонируйте репозиторий
```bash
git clone <repo_url>
cd <repo_directory>
```

### 2. Создайте виртуальное окружение
```bash
python -m venv .venv

# Windows
.venv\Scripts\activate

# Linux/macOS
source .venv/bin/activate
```

### 3. Установите зависимости
```bash
pip install -r requirements.txt
```

### 4. Скопируйте и заполните .env файл
```bash
cp .env.example .env
```

Отредактируйте `.env` и заполните:
- `BOT_TOKEN` — токен бота от @BotFather
- `ADMIN_ID` — ваш Telegram ID
- `CLAUDE_API_KEY` — ключ OpenRouter (обязательно)

### 5. Запустите бота
```bash
python main.py
```

## Развёртывание на Railway.com

Railway — это облачная платформа для быстрого развёртывания приложений. Бот автоматически переключится на режим webhooks на Railway.

### Быстрое развёртывание

1. **Создайте аккаунт на [Railway.com](https://railway.app/)**

2. **Подключите ваш GitHub репозиторий**
   - Нажмите "New Project" → "Deploy from GitHub"
   - Выберите репозиторий с ботом

3. **Задайте переменные окружения в Railway dashboard**
   - Перейдите в Variables проекта
   - Добавьте все переменные из `.env.example`:
     ```
     BOT_TOKEN=your_token
     ADMIN_ID=your_id
     CLAUDE_API_KEY=your_key
     ENVIRONMENT=production
     ```
   - Также добавьте другие API ключи если используются (OPENAI_API_KEY, GROQ_API_KEY)

4. **Railway автоматически:**
   - Установит зависимости из requirements.txt
   - Запустит команду из Procfile
   - Генерирует RAILWAY_PUBLIC_DOMAIN
   - Бот получит webhook URL из RAILWAY_PUBLIC_DOMAIN

5. **Бот начнёт работать!**
   - Railway автоматически предоставит доменное имя
   - Webhook автоматически установится
   - Проверьте логи в Railway dashboard

### Мониторинг и логи

В Railway dashboard вы можете:
- Просмотреть логи работы в реальном времени
- Перезапустить приложение
- Изменить переменные окружения
- Просмотреть использование ресурсов

### Проблемы при развёртывании

**Проблема: "Failed to set webhook"**
- Убедитесь, что `BOT_TOKEN` правильный
- Проверьте, что `RAILWAY_PUBLIC_DOMAIN` установлен
- Подождите 1-2 минуты после развёртывания

**Проблема: "DatabaseError"**
- БД автоматически создаётся при первом запуске
- Проверьте, что у приложения есть права на запись

**Проблема: "API KEY missing"**
- Проверьте, что `CLAUDE_API_KEY` установлен в переменных Railway

## Технологии

- **aiogram 3** — фреймворк для Telegram ботов
- **aiohttp** — асинхронный HTTP клиент и веб-сервер
- **SQLite** — база данных
- **OpenRouter** — доступ к Claude/GPT моделям
- **OpenAI Whisper / Groq** — транскрипция аудио
