# EnglishDV Telegram Bot

Телеграм-бот для изучения английского языка с AI-проверкой домашних заданий и аудио-уроками.

## Структура проекта

```
тг бот/
├── main.py              # Главный файл бота (точка входа)
├── requirements.txt     # Зависимости Python
├── .env.example        # Пример переменных окружения
└── src/
    ├── __init__.py      # Пакет бота
    ├── config.py        # Конфигурация (токены, API ключи)
    ├── database.py      # Работа с SQLite БД
    ├── lessons.py       # Данные 25 уроков
    ├── keyboards.py     # Клавиатуры бота
    ├── ai.py            # AI промпты и функции
    └── reading_texts.py # Тексты для аудио-уроков
```

## Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Скопируйте `.env.example` в `.env` и заполните переменные:
   - `BOT_TOKEN` — токен бота от @BotFather
   - `ADMIN_ID` — ваш Telegram ID
   - `CLAUDE_API_KEY` — ключ OpenRouter (обязательно)

## Запуск

```bash
python main.py
```

## Функции бота

- 📚 **25 уроков** английского языка (A1-B2)
- ✍️ **Проверка ДЗ** через AI (текст и фото)
- 🎙 **Аудио-уроки** — чтение вслух с оценкой произношения
- 💬 **Чат с AI-учителем**
- 📖 **Словарь** с интервальным повторением
- 📆 **Расписание** уведомлений
- 🏆 **Рейтинг** учеников

## Модули

| Файл | Назначение |
|------|------------|
| `config.py` | Конфигурация и переменные окружения |
| `database.py` | Все операции с SQLite (пользователи, словари, история) |
| `lessons.py` | Тексты 25 уроков (теория, ДЗ, тесты) |
| `keyboards.py` | Клавиатуры (главное меню, урок, расписание) |
| `ai.py` | AI промпты, парсинг ответов, транскрипция аудио |
| `reading_texts.py` | Тексты для аудио-уроков с фокусами и подсказками |
| `main.py` | Обработчики команд и состояния FSM |

## Требования

- Python 3.10+
- aiogram 3.x
- aiosqlite
- httpx
